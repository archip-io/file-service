# Микросервис управления файлами

Добро пожаловать в репозиторий File Service! File Service - это микросервис, который отвечает за управление файлами (
загрузка, получение, удаление).

## Технологии

- Java 17
- Spring Boot 3
- PostgreSQL
- Yandex Disk
- RabbitMQ
- Maven
- Docker
